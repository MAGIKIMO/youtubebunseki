# YouTube Comment Sentiment Lab

YouTube動画のコメントを取得し、AI感情分析モデルを用いて、視聴者の反応、炎上リスク、主要キーワードを可視化するStreamlitダッシュボードです。

単なる感情分析ツールではなく、複数動画の比較、時間帯別の感情変化、否定的コメントのモニタリングを一つの画面で確認できるように構成しました。

## 開発背景

YouTube動画では、視聴者の反応がコメント欄に集まります。しかし、コメント数が多い場合、手作業で全体の反応や否定的な意見の増加を把握するには時間がかかります。

そこで、YouTube Data APIを用いてコメントを取得し、AI感情分析モデルで分類・可視化することで、動画に対する反応やリスクを短時間で把握できる分析ツールを開発しました。

---

## 主な機能

- **AI感情分析**  
  Transformersベースのモデルを用いて、コメントを「賛成・支持」「反対・批判」「質問・情報」「ユーモア・皮肉」「中立・その他」に分類

- **複数動画比較**  
  複数のYouTube URLを入力し、動画ごとのコメント反応や危険度を比較

- **危険度モニタリング**  
  否定的コメントや皮肉・ユーモアコメントの割合をもとに危険度スコアを算出し、警告を表示

- **時間帯別トレンド分析**  
  コメントの感情変化を時系列グラフで可視化し、簡易的な将来傾向も表示

- **キーワード分析**  
  全体コメントおよび否定的コメントに含まれる主要キーワードを抽出し、棒グラフとワードクラウドで表示

- **Streamlit UI**  
  Streamlitを用いて、分析結果をダッシュボード形式で確認できるように実装

本プロジェクトでは、主に以下の実装を担当しました。

- コメント分類結果の可視化
- 危険度スコア計算

## 実行方法

```bash
pip install -r requirements.txt
streamlit run app.py
```

Windows環境で `streamlit` コマンドが認識されない場合は、以下のように実行します。

```bash
py -m streamlit run app.py
```

---

## 使用方法

1. Google Cloud ConsoleでYouTube Data API v3のAPI Keyを発行します。
2. Streamlit画面のサイドバーにAPI Keyを入力します。
3. 分析したいYouTube動画URLを入力します。
4. 分析するコメント数を選択します。
5. 分析開始ボタンを押します。
6. 感情分類結果、危険度スコア、時系列トレンド、キーワード分析を確認します。

---

## 感情分類カテゴリ

- **賛成・支持**  
  肯定的な反応

- **反対・批判**  
  否定的な反応

- **質問・情報**  
  質問や情報要求

- **ユーモア・皮肉**  
  ユーモア、皮肉、感嘆表現

- **中立・その他**  
  明確な感情に分類しにくいコメント

---

## 危険度基準

- **50%以上**  
  危険。早急な確認・対応が必要な可能性があります。

- **30〜50%**  
  注意。継続的なモニタリングが必要です。

- **30%未満**  
  比較的安定した状態です。

---

## 工夫した点

AIモデルが正常に利用できない場合でも分析が止まらないよう、複数モデルのfallback処理とキーワードベースの分類処理を実装しました。

また、単純な感情分析で終わらせず、危険度スコア、複数動画比較、時系列トレンド、キーワード分析を組み合わせることで、視聴者の反応をより実務的に把握できる構成を意識しました。

## 学んだこと

この開発を通じて、外部APIからのデータ取得、AIモデルの組み込み、データ可視化、ユーザーが理解しやすいUI設計を一貫して経験しました。
